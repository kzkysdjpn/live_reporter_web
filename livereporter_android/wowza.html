Wowza Media Server Setup

Return to Main View

This document is Wowza Streaming Server setup for Live-Reporter.

Enter to “Server” for publisher setup.
Add to new publisher. If any publisher exist, skip to this process.
Enter to “Application->live->Incoming Streams” for checking streams.

The “live” is application name. This application name exist as wowza default install.

Input wowza media server information on Live-Reporter App.

In this sample, title field are input as “live/test”.

These “live” and “test” are application name and streaming name, respectively.

The application name should be same to the wowza media server setting.

The streaming name is able to your arbitrary definition.

Tap to “Connect” button. And then input wowza authentication on Live-Reporter App.

These user ID and password are publisher information on wowza media server.

Check “Incoming Streams” on the Wowza Media Server web page.

Your arbitrary streaming name may exist on “Incoming Streams” list.
