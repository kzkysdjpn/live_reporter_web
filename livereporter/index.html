Live-Reporterについて

Live-ReporterはiOS上からカメラ映像を伝送するアプリケーションです。

伝送する映像を確認するには、PC/スマートフォンアプリを使って受信します。

また、オープンソースもしくは有料の配信サーバーへ向け映像をプッシュ送信することができます。

本ページでは、基本的な使用方法の説明を記載しています。

その他のボタンについて

クライアントアプリを使った伝送

クライアントアプリは代表的なものとしてVideoLANプロジェクトのVLCプレーヤーがあります。

アプリ起動後、下記の画面が表示されます。画面下部に表示されたアドレスを確認してください。

iPhone/iPodTouchに複数のIPアドレスが与えられている場合、表示されているアドレスをタップすることで、

他に与えられているアドレス情報を表示することができます。


PCもしくはスマートフォン側にてVLCプレイヤーを起動します。

「ファイル」->「ネットワークを開く」を選択します。

「ソースを開く」のダイアログが表示されます。

「URL」の欄に先ほどアプリにて表示されたアドレスを入力し、「開く」ボタンを押します。

サーバーへのプッシュ伝送

本アプリが対応するプッシュ伝送できる配信サーバーは、

Apple製DarwinStreamingServer、

Wowza Media Server

などがあげられます。

アプリ起動画面にて、上部に表示されたアドレス入力欄にサーバーアドレスを入力し、接続ボタンを押してください。

各サーバーの設定方法については、サポートページの方に説明をご参照ください。
