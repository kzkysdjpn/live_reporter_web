Darwin Streaming Server Setup

Return to Main View

This document is Darwin Streaming Server setup for Live-Reporter.

Enter to “General Settings” for “Broadcast Username/Password”.
Input your arbitrary “Username” and “Password”.
On Live-Reporter, these input values will be requested for broadcast authentication.
Input Darwin Streaming Server information on Live-Reporter App.

In this sample, title field are input as “test.sdp”.

“test.sdp” is streaming name.

The streaming name is able to your arbitrary definition.

Tap to “Connect” button. And then input Darwin Streaming Server authentication on Live-Reporter App.

These user ID and password are “Broadcast Username/Password” on Darwin Streaming Server.

Check “/usr/local/movies” of the file system on the server.

Your arbitrary streaming name file may exist as “test.sdp”.
